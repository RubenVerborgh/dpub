Examples For the Task Force Activities

  • Layout and Styling (aka Pagination)
    • requirements for print and digital publishing from HTML and CSS using the Latin alphabet
    • looking at standard ways to implement pagination
  • Content and Markup
    • provide a standard way to add a logical structure on top of HTML
    • working with a W3C Working Group that develops similar structures for accessibility
  • Identification
    • work on the general principles of identifying a document as well as its content
